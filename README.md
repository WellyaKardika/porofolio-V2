# Wellya Kardika Portfolio V2
portfolio built with React and Vite. Features a brutalist monochrome design, smooth scroll animations, and interactive typography.

## ✨ Features

- **Monochrome Aesthetic**: Deep black background (`#0C0C0C`) with bright white/silver text and global grayscale images.
- **Cinematic Animations**: Powered by Framer Motion (`RevealText`) and GSAP for fluid navigation.
- **Interactive UI**: Custom magnetic elements and bottom-to-top liquid hover effects on buttons.
- **Responsive Layout**: Flawless display across mobile, tablet, and desktop devices.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/WellyaKardika/porofolio-V2.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
