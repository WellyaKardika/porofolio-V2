# Wellya Kardika Portfolio V2

A modern, highly interactive portfolio built with React, Vite, and TypeScript. It features a brutalist monochrome design, smooth scroll animations, and interactive typography that brings the user experience to life.

## ✨ Features

- **Monochrome Aesthetic**: Deep black background (`#0C0C0C`) with bright white/silver text and global grayscale imagery for a premium, cinematic feel.
- **Cinematic Animations**: Powered by [Framer Motion](https://www.framer.com/motion/) (`RevealText`) and [GSAP](https://gsap.com/) for fluid page transitions and reveal effects.
- **Smooth Scrolling**: Implemented with [Lenis](https://lenis.studiofreight.com/) for a buttery smooth scrolling experience.
- **Interactive UI**: Custom magnetic elements and bottom-to-top liquid hover effects on buttons.
- **Responsive Layout**: Flawless display across mobile, tablet, and desktop devices utilizing [Tailwind CSS](https://tailwindcss.com/).
- **Modern Routing**: Fast and seamless client-side routing with React Router DOM.

## 🛠️ Tech Stack

- **Core**: [React](https://reactjs.org/) 19, [TypeScript](https://www.typescriptlang.org/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (with `clsx` and `tailwind-merge` for utility class management)
- **Animations & Effects**: [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/), [Lenis](https://github.com/darkroomengineering/lenis)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Analytics**: Vercel Speed Insights

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/WellyaKardika/porofolio-V2.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd porofolio-V2
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```

## 🌐 Deployment

This project is optimized for deployment on [Vercel](https://vercel.com) (including speed insights) or any other static hosting platform. Simply push to your repository or use the Vercel CLI to deploy.
